# Digital Wallets, Bills & Transactions Management System

DWBTMS (Digital Wallets, Bills & Transactions Management System) is a system designed to reduce the red tape work<br>
  by automating the processes of billing and financial transactions using C# programming language withing the ASP.NET Core environment.<br>
  the project main purpos is to practice clean code architecture, some important and related design patterns<br>
  SOLID principles, and of course OOP best practices.<br>
  Also it shows the importance and power of using middlewares (for exception handling in this case), and the beauty<br>
  of using Swagger annotation to document REST APIs.

## Tech Stack

- **.NET 10.0**
- **ASP.NET Core Web API**
- **C# 13**
- **ADO.NET**
- **Microsoft SQL Server**
- **Mediator Pattern**
- **Unit Of Work Pattern**
- **Dependency Injection**
- **Swagger**
- **Clean Architecture**
- **SOLID Principles**

## Local Development

To run this project locally:

1. Clone the repo using the command: `git clone https://github.com/Saeed-AbuYasser/Digital Wallet.git`
2. Restore the database: `Digital Currency.bak`
3. Open `https://localhost:7113/swagger`
